What it does

Everything a student needs, automated

Lecture Summarizer turns passive slide decks into an active learning experience powered by AI.

📊

Slide Summarization

Upload any .pptx and get bullet-point summaries for every slide using a HuggingFace DistilBART model.

💬

Contextual Q&A

Ask questions about any slide or the full lecture. The AI answers using only your lecture content — no hallucinations.

📝

Quiz Generation

Auto-generate multiple-choice and short-answer quizzes based on the lecture material to test comprehension.

📘

Assignment Creation

Generate structured university-level assignments with short answer, analytical, and application questions.

🗂️

Course Management

Organize summaries by course. Save your lecture history, revisit past summaries, and track assignments and quizzes.

🔒

Secure Auth

JWT-based authentication with bcrypt password hashing. Guest mode available for quick use without an account.

Technical Report — STAR Format

Challenges I faced

A structured reflection on the key technical and design challenges encountered during development.

S

Situation

Students often receive large PowerPoint lecture files but lack time to read every slide thoroughly. Existing tools either required expensive API subscriptions or produced low-quality summaries that missed key concepts. I needed a self-hosted solution that could intelligently extract and condense lecture content.

T

Task

Build a full-stack AI application that extracts text from .pptx files, summarizes each slide using a local AI model, supports contextual Q&A grounded in the lecture content, and allows users to organize summaries by course — without requiring an external API key.

A

Action

Integrated HuggingFace's DistilBART model with careful token-length management to avoid truncation errors. Built a keyword-based slide retrieval system so the Q&A model receives only the most relevant context. Implemented dual-mode auth (guest vs. logged in) and a session persistence layer using PostgreSQL that survives server restarts.

R

Result

The application successfully summarizes multi-slide presentations in under 30 seconds locally, generates relevant quizzes and assignments, and persists all user data reliably. The modular design allowed easy integration of an OpenAI fallback without changing core application logic.
